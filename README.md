
# Driving Free Academe | أكاديمية القيادة الحرة

أول أكاديمية رقمية متخصصة في تعليم القيادة في دبي بأسلوب أكاديمي وعلمي، مصممة وفق معايير هيئة الطرق والمواصلات (RTA).

## النطاق المربوط
[drivingfreee.online](https://drivingfreee.online)

## المميزات الرئيسية
- **منهج أكاديمي متكامل:** مراحل تعليمية شاملة من الصفر للاحتراف (مجانية بالكامل).
- **مكتبة علمية:** دراسات تقنية حول فيزياء القيادة وسيكولوجية السائق.
- **محاكي RTA الذكي:** مولد أسئلة مدعوم بالذكاء الاصطناعي (Genkit) يحاكي الاختبارات الرسمية.
- **أمان وحماية:** محمي بواسطة Google reCAPTCHA Enterprise.
- **مراقبة الأداء:** متصل بـ Firebase Performance Monitoring.

## إرشادات النشر (Deployment)

### 1. النشر على قناة تجريبية (Preview Channel)
لاختبار التغييرات قبل اعتمادها على الموقع الحي، استخدم الأمر التالي:
```bash
firebase hosting:channel:deploy preview_name
```
سيعطيك هذا الأمر رابطاً مؤقتاً لمراجعة الموقع.

### 2. النشر النهائي (Production)
بعد التأكد من سلامة التغييرات، قم بالنشر على القناة الرئيسية:
```bash
firebase deploy --only hosting
```

## ملاحظات الربط النهائي
1. تم إعداد `next.config.ts` للسماح بجميع النطاقات المستخدمة للصور (ibb.co, unsplash, etc.).
2. تأكد من تفعيل سجلات DNS (A record و CNAME) في لوحة تحكم مزود النطاق الخاص بك لتوجيهها إلى Firebase.
