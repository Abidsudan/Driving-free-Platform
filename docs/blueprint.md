# **App Name**: Driving Free Academe

## Core Features:

- Comprehensive Content Display: Showcase structured educational content across various sections including Home, Academic Curriculum (4 stages), Scientific Library, and Rules & Signs, utilizing an academic and scientific tone in Arabic.
- Dynamic Knowledge Assessment Tool: Provide an interactive, RTA-simulated quiz experience with real-time scoring (e.g., 'معدل الإدراك المعرفي'). Questions are generated as an AI tool to test driving scenarios and rule comprehension, with instant feedback on correct/incorrect answers.
- Premium Content Gating: Present blurred or locked premium sections with clear indications (e.g., a lock icon) and a call to action for subscription, without displaying pricing.
- Responsive Navigation System: Implement an intuitive and responsive navigation bar that adapts to screen size, appearing as a fixed top bar on desktops and a fixed bottom bar on mobile devices for easy access to all main sections.
- Expert Experience Showcase: Prominently display the instructor's 8+ years of field experience in Dubai driving to build trust and credibility, especially on the Home page.
- Regulatory Content Integration: Integrate accurate information from official sources such as the RTA Driving Manual, DSSSM rules, immediate practical test failures, and theoretical test questions.

## Style Guidelines:

- Dark color scheme with a deep blue-tinted background (#14191E) to foster concentration and professionalism. Primary actionable elements (buttons, links) are a sophisticated medium blue (#4666BF). Headlines and premium features are highlighted with a distinguished gold accent (#E0BA1B), reflecting quality and prestige.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, technically precise, and academic feel. Body font: 'Inter' (sans-serif) for high readability in longer textual content. Code font: 'Source Code Pro' (monospace) for clear display of technical code examples.
- Use clear, academic-style vector icons (e.g., from Lucide React) for each section and content card, enhancing visual guidance without clutter, maintaining a modern and clean aesthetic.
- Clean, structured, and responsive layouts that prioritize content readability. A flexible grid system is used for content presentation, adapting seamlessly across desktop and mobile devices. Ample white space (dark space, in this case) is utilized to prevent visual fatigue.
- Subtle, non-distracting fade-in animations for section transitions and content loads to provide a smooth, modern user experience, avoiding abrupt visual changes.
